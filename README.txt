PROYECTO: PETOTECH – SISTEMA DE JUEZ ELECTRÓNICO

Descripción:
Aplicación móvil desarrollada en Kivy que permite a jueces
asignar puntuaciones en tiempo real mediante WebSockets,
sin sustituir el criterio humano.

Requisitos:
- Python 3.10 o superior
- Instalación de dependencias: pip install -r requirements.txt

Ejecución:
- Ejecutar run.bat
- Ingresar contraseña del combate
- Seleccionar juez
